Executive Summary

This project develops a modular, open-source GUI application for an innovative multispectral vision-based apple sorting system. The system integrates a JAI 3-sensor camera with AI models for automated fruit quality inspection, grading, and sorting in real-time.

Key Innovation

Unlike proprietary industrial systems, this GUI is open-source, modular, and designed for rapid technology iteration, addressing a critical software gap in automated agricultural inspection systems.

Core Capabilities

  • Real-time multispectral imaging (3 customizable wavebands)
  • Dynamic AI model switching and comparison
  • Automated sorting based on quality grading
  • Comprehensive data visualization and logging
  • Production-ready reliability and performance

Key Features & Innovation

Dynamic Model Switching

Problem: Traditional systems hardcode AI models

Solution: Runtime model loading and switching

Benefits: Compare models, A/B testing, rapid iteration

Perfect Multispectral Sync

Advantage: Hardware-synchronized via prism

Benefit: No software sync complexity

Result: All 3 sensors capture simultaneously

Modular Architecture

Design: Plug-and-play components

Flexibility: Swap camera, change hardware easily

Community: Clear APIs, easy to extend

Open Source

Transparency: Fully open code

Reproducibility: Research-grade quality

Community: Free to use and modify
